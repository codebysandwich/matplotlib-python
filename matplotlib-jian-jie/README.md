# matplotlib简介

`matplotlib`是建立在`numpy`基础上的多平台可视化程序库。它以多种拷贝格式和跨平台的交互式环境生成出版物质量的图形。

## 安装

在终端中执行以下语句即可完成安装
```shell
pip install matplotlib
```

在python交互环境下执行
```python
>>> import matplotlib as mpl
>>> mpl.__version__
'3.1.2'
```
执行输出的版本号因人而异, 个人建议安装matplotlib之前先安装`numpy`和`pandas`。均可以通过`pip`执行安装。
